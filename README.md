### Chat modo consola hecho en GO con libreria: github.com/VladimirMarkelov/clui

### Comandos para: Windows (usando Go)
```batch
go install gitlab.com/RicardoValladares/chat@latest
cd %GOPATH%/bin
chat.exe
```

### Comandos para: BSD, Linux y Mac (usando Go)
```bash
go install gitlab.com/RicardoValladares/chat@latest
cd $GOPATH/bin
./chat
```
